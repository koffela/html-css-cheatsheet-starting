HTML/CSS Reference

Common Block Tags

Tag Name Description
<div> Div Used as a container for other HTML elements.
<ul> Unordered List Creates a list of items, each one with a bullet point
<ol> Ordered List Creates a list of items, each one with a number or letter

Common Block Attributes

Attribute Name Description
inline-block Display: Inline-Block Display items next to each other (commonly used for navigation)
